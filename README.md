# InterviewPilot AI

InterviewPilot AI is an AI-powered Interview Coach that helps job seekers prepare for interviews through resume analysis, job matching, personalized mock interviews, and actionable feedback.

## Features

- Resume Upload & Analysis
- ATS Score Prediction
- Resume ↔ Job Matching
- Personalized Interview Questions
- AI Answer Evaluation
- Interview Readiness Dashboard

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python
- PostgreSQL

### AI
- OpenAI API
- Specialized AI Agents
